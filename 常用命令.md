以下是生成nupkg文件的具体命令：
1. 生成所有项目的nupkg文件 ：
   dotnet pack -c Release -o nupkg
2. 生成单个项目的nupkg文件 （例如WellTool.Core）：
   dotnet pack WellTool.Core\WellTool.Core.csproj -c Release -o nupkg
3. 生成特定版本的nupkg文件 ：
   dotnet pack WellTool.Core\WellTool.Core.csproj -c Release -o nupkg /p:Version=8.0.1
4. 生成WellTool.All的nupkg文件 ：
   dotnet pack WellTool.All\WellTool.All.csproj -c Release -o nupkg
